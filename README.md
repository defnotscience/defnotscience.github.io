### Server Deployment

> [!IMPORTANT]
> You **cannot** deploy to static web hosts, including Netlify, Cloudflare Pages, and GitHub Pages.

You must run these commands on your server:  
`git clone https://github.com/defnotscience/defnotscience.github.io`  
`cd Cosmic`  
`npm install`  
`npm start`
